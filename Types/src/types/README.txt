[데이터 타입]

(원시타입)
1. cpu를 반영한 타입
2. 3가지 타입
   - 정수
   - 실수
   - 논리
   
3. 메모리는 바이트 단위
   - 1바이트는 8비트
   - 메모리를 나눌 때 짝수로 분할해서 사용
   - 31Byte -> 32Byte, 32비트 O/S이상에서는 4Byte 단위로 메모리를 분할해서 사용 
   
4. 정수
   - byte : 1 byte
   - char : 2 byte
   - short : 2 byte
   - int : 4 bytge
   - long : 8 byte
