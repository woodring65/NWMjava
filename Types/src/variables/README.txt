[Variable] 변수

(변수)
1. 변하는 수를 담을 수 있는 그릇
2. 하나의 값을 저장할 수 있는 메모리 공간
3. 메모리 번지에 대한 별칭


(변수 작성 규칙)
1. 영문자, 숫자, 특수문자($, _)의 조합 : a123, $abc, _abc, $124, _123
2. 첫번째 문자는 숫자가 나와서는 안된다. : 1abc(x)
3. 영문 대소문자 구별 : (abc, Abc) 서로 다른 변수
4. 길이의 제한은 없다.
5. 자바 예약어(Reversed Word) : 교재(P31)
6. 예제
   예1) a, b, abc, ab123
   예2) $abc, $123, _123
   예3) $a123
 

(변수의 선언)
타입 변수이름
타입 변수이름=초기값

정수 변수이름
정수 변수이름=정수

(변수타입)
정수, 실수, 논리


(할당 연산자, =)
정수(10)을 변수(a)에 할당 즉 저장한다.
오른쪽의 표현식(연산식)이나 처리결과를 왼쪽의 변수에 할당(저장)
int a = 10; // int a <- 10, 10 -> a 

(리터럴, Literal)
정수 리터널, 실수 리터널, 문자 리터널, 논리 리터널
상수(constant)의 의미를 가지고 있지만 상수와는 구별된다.
예) 10, -5, 0.34, "hello"
 

(원시타입) 
primitive type
boolean < short(2byte), char(byte) < int(4byte) < float(4byte) < double(byte)
