Hello.java

1. 자바 프로그램의 학습을 위한 안내 샘플이다.

(eclipse)
자바 개발을 위한 IDE(통합개발환경)
실행: Ctrl + F11

(Java Language)
1. 주석: 주석을 코드의 실행에 영향을 주지않는 설명문
      - 블럭주석(/* ... */)은 여러 라인의 범위에 주석
      - 라인주석(// ... )은 //부터 시작해서 해당 라인의 끝까지 주석 
      예1) 블럭주석: /* 블럭주석입니다. */
      예2) 라인주석: // 여기서부터 주석입니다.

2. 소스파일이름은 클래스(class) 이름과 동일해야 한다.
   - 파일이름 : Hello.java
   - 클래스이름 : class Hello     
   - 파일이름(Hello)와 클래스 이름(Hello)

3. 모든 문장(statement)의 끝은 세미콜론(;)으로 끝나야 한다.

(System.out.println)
Prints a String and then terminate the line.
java.lang.System.out.println(String x);   

String: 문자열을 처리하는 클래스
문자열: 문자들의 모임인데 종료(terminate) 문자를 가지고 있다.
