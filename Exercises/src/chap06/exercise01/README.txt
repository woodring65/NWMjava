[문제1] 객체와 클래스에 대한 설명으로 틀린 것은?
정답: 3번
해설: 하나의 클래스로 다수의 객체를 만들 수 있다.

1. 클래스는 객체를 생성하기 위한 설계도(청사진)와 같다.
2. new 연산자로 클래스의 생성자를 호출함으로써 객체가 생성된다.
3. 하나의 클래스로 하나의 객체만 생성할 수 있다.
4. 객체는 클래스의 인스턴스이다.

class Abc {
	Abc() {}
	Abc(int x, int y) {}
}

Abc a1 = new Abc();
Abc a2 = new Abc();
Abc a3 = new Abc();
Abc a4 = new Abc(10, 20);
